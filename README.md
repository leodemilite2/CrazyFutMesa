# Crazy FutMesa

Crazy FutMesa é um jogo casual de física inspirado em Golf Battle, utilizando a temática do futebol de mesa.

## Objetivo

Criar um jogo divertido baseado em física, acessível para qualquer pessoa, mesmo sem conhecer futebol de mesa.

## Engine

- Unity 6

## Plataforma

- Android
